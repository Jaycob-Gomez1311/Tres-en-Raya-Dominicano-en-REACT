## 🧩 Descripción del juego

Este proyecto consiste en una versión personalizada del clásico juego **Tres en Raya (Tic-Tac-Toe)**.

La idea está inspirada en los juegos que solía jugar en la escuela y en casa cuando no había internet o se iba la luz. En lugar de fichas tradicionales, utilicé **tapas de refresco**, que representan a los dos equipos:

* 🔴 Guerra (tapas rojas)
* 🔵 Paz (tapas azules)

El juego incluye:

* Modo 2 jugadores (en el mismo dispositivo)
* Modo contra la máquina (IA básica)
* Sistema de turnos con lanzamiento de moneda
* Marcador persistente
* Efectos de sonido

Aunque no se pudo implementar la versión en modo asterisco, el resultado final cumple con una experiencia visual y funcional bastante completa.

## ⚠️ Qué fue lo más difícil

Una de las partes más complicadas fue intentar implementar el modo en **asteriscos**, ya que afectaba directamente la lógica ya construida del juego, por lo que se decidió no incluirlo.

También los sonidos presentaron cierta dificultad. Aunque funcionan, pueden sentirse algo largos o poco sincronizados. Aun así, quise incluir elementos representativos como:

* El sonido de fichas tipo dominó
* Celebración con bebida (cerveza)
* Sonido de burla ("chuipi") al perder

Esto fue pensado para darle más personalidad y ambiente al juego.

## 🚀 Cómo ejecutar el proyecto

1. Clonar el repositorio
2. Instalar dependencias:

   ```bash
   npm install
   ```
3. Ejecutar en modo desarrollo:

   ```bash
   npm run dev
   ```
